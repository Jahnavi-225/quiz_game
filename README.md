Python Quiz Game
Description

A console-based quiz application built using Python that presents multiple-choice questions, validates user input, and provides real-time feedback with final score tracking. The project is designed using object-oriented programming principles for better structure and maintainability.

Features
Randomized question order for each run
Input validation to handle incorrect entries
Immediate feedback on answers
Final score calculation
Tech Stack
Python
Random module
Project Structure
main.py  
quiz.py  
questions.py  
How to Run
python main.py
Implementation Details

The application is built using a modular approach:

A Question class to store question data and validate answers
A Quiz class to manage quiz flow, scoring, and user interaction
Questions are shuffled using the random module to ensure variation
Purpose

This project demonstrates the use of object-oriented programming, control flow, and input handling in Python through a simple interactive application.
